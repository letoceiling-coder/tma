# Настройка каналов для подписки

## Текущие каналы

После выполнения команды `php artisan wow:setup-channels` будут настроены следующие каналы:

1. **@neeklo_studio** - Neeklo Studio (приоритет 1)
2. **@neiroitishka** - Neiroitishka (приоритет 2)

## Команда для настройки

```bash
php artisan wow:setup-channels
```

Эта команда:
- Удаляет все существующие каналы
- Добавляет каналы, указанные в команде
- Показывает список активных каналов

## Важно

1. **Бот должен быть администратором всех каналов** - иначе проверка подписки не будет работать
2. **Каналы должны быть публичными** или бот должен иметь доступ к методу `getChatMember`
3. После настройки каналов нужно **пересобрать фронтенд**, если каналы указаны там вручную

## Обновление каналов в коде

Если нужно добавить/изменить каналы:

1. **Backend (БД):** Обновите команду `app/Console/Commands/SetupChannels.php`
2. **Frontend:** Обновите массив `CHANNEL_USERNAMES` в `frontend/src/App.tsx`
3. **Пересоберите фронтенд:** `npm run build:react`
